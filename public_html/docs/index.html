    معرفی

    این داکیومنت برای تامین تمامی اطلاعاتی میباشد که شما برای کار با API این وبسایت نیاز دارید.
    تمامی مسیرها به همراه توضیحات و پارامترهای لازم و همچنین ریسپانس هایی که برگشت داده خواهند شد نوشته شده اند.
    هر مسیر یک مثال نیز دارد که در صورت موفقیت آمیز بودن درخواست برگشت داده خواهد شد.
    مهم
    تعاریفی که برای کار با api نیاز دارید:
    توکن:رشته ای میباشد که پس از لاگین کردن کاربر به فرانت برگشت داده میشود و تمامی احراز هویت ها بر اساس همین توکن انجام خواهند شد.
    در مسیرهایی که به رنگ قرمز نوشته شده است که این مسیر به احراز هویت نیاز دارد حتما باید توکن به همراه هدرها بصورت زیر ارسال شود:
    کلید: Authorization
    مقدار: {توکن مورد نظر} Bearer
    در اکثر موارد فرمت برگشت داده ها به این صورت است که داخل یک آبجکت با کلید data برگشت داده میشوند ولی در دو یا سه مورد فرمت فرق میکند که براش مثال زده شده است.
    به فرض مثال برای دریافت 4 ویلای محبوب،تمامی دیتا دارای آبجکت data هستند و هر کدام از ویلاها داخل یک آبجکت دیگر هستند که از طریق آن میتوان به مشخصات کلی ویلا،جزییات ویلا و هزینه های ویلا دسترسی داشته باشید که هر کدام کلید مخصوص خود را دارند و نام گذاری کلیدها بصورتی میباشد که میتوان تشخیص داد که کاربرد هر کدام چی میباشد.
    هر چند تمامی مسیرها همراه با مثال و توضیحات کامل میباشند
